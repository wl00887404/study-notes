# C++ Self Teaching

## CMAKE

> 功用貌似是「產生自動產生 makefile 」

### 必備

- cmake_minimum_required
- project
- add_executable

## C++

```bash
sudo apt-get install build-essential

```

```bash
sudo apt-get install freeglut3-dev
```
